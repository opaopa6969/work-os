# Bug Hunter
A template specialized in investigating and fixing issues within an existing codebase.

## Contents
- Thorough root cause analysis.
- Creation of automated tests (unit and regression tests).
- Refactoring to prevent recurrence.

## Agent Workflow
1. Analyze reported issues and investigate the code.
2. **Before fixing**, always write a test case that reproduces the bug.
3. Apply the fix and verify that the test passes.
4. Run regression tests to ensure no side effects on related code.
