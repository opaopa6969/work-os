# Advanced Development Template (Pro+)
The ultimate project starter kit covering Specs, Architecture, Permissions, and Release Flow.

## Contents
- `docs/specs.md`: Functional definitions (What to build).
- `docs/architecture.md`: Technical design (How to build, data flow, components).
- `docs/permissions.md`: Permission management (Auto-approval scope, restrictions).
- `docs/backlog.md`: Task tracking.
- `CHANGELOG.md`: Version history.
- `AGENT.MD`: Master instructions for autonomous agent behavior.

## Agent Workflow
1. Organize requirements in `specs.md` and define implementation strategy in `architecture.md`.
2. Check `permissions.md` to understand the boundaries of autonomous actions.
3. Proceed with implementation using `backlog.md` while following Conventional Commits.
