# 司令官 (Commander) テンプレート
複数のエージェントを束ね、プロジェクトの最高意志決定を行うためのテンプレートです。

## 内容
- `docs/fleet.json`: 配下のエージェント（Worker）の定義。
- `AGENT.MD`: 司令官としての振る舞いと、ユーザー・Workerとの連携ルール。

## エージェントの動き
1. ユーザーの意図を汲み取り、タスクを分割する。
2. 適切な Worker エージェントに指示を出し、その進捗を監督する。
3. Worker が助けを求めた際、または入力待ちになった際、解決策を提示する。
4. 重要な局面ではユーザーに直接確認を求める。
